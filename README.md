## Description

Genius is a Progressive Web App (PWA) for trivia enthusiasts. Swipe through flashcards, test your knowledge, and track your progress - all offline-first!

## Features

- [x] React 18 + Vite + TypeScript foundation
- [ ] PWA with offline support
- [ ] Swipeable flashcards with physics
- [ ] 3D card flip animations
- [ ] Local score storage (IndexedDB)
- [ ] Installable on mobile

## Tech Stack

- **Core**: React 18 + Vite + TypeScript
- **PWA**: vite-plugin-pwa + Workbox
- **UI/Motion**: Framer Motion + Tailwind CSS
- **Data**: Dexie.js (IndexedDB) + OpenTDB API
- **Quality**: ESLint + Prettier + Playwright

## Installation
